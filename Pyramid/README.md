# Pyramid Learning Guide

This repository contains beginner-friendly notes for learning the Pyramid web framework step by step.

If you want to learn Pyramid, follow the files in the given order and read them one by one.

## Learning Order

1. `1_Architecture_Overview.md`  
   Learn the basic architecture of Pyramid and how the request-response cycle works.

2. `2_Setup_and_Installation_of_Pyramid.md`  
   Learn how to set up Pyramid in your development environment.

3. `3_Creating_Routes_and_Views.md`  
   Learn how routes and views work in Pyramid.

4. `4_Database_Integration.md`  
   Learn how to connect a database with Pyramid for data persistence.

5. `5_Authentication_and_Authorization.md`  
   Learn how Pyramid handles user login, permissions, and access control.

6. `6_Rest_topics.md`  
   Learn the remaining important topics such as templates, static files, forms, sessions, project structure, and error handling.

## Recommended Approach

Read each file carefully before moving to the next one.  
Do not start a project until you understand the basic concepts clearly.

## Goal

The goal of this learning guide is to build a strong foundation in Pyramid before starting a real web development project.
